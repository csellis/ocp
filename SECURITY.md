# Security

Report security issues to me@chrisellis.dev. Do not open a public issue.

There is no bug bounty. Reports are acknowledged within seven days. Please give a reasonable disclosure window before going public.

OCP runs locally and reads files in the watched repo. It does not exfiltrate data. The hosted (Mode B) surface described in `docs/ARCHITECTURE.md` is not yet built; this policy will be updated when it is.
