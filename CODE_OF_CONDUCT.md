# Code of Conduct

OCP adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct. The full text is canonical; this file points to it and adds project-specific notes.

## In short

- Be respectful. Disagree on the work, not on the person.
- No harassment in any form, on any project surface (issues, PRs, commit messages, project chat, Discord, Slack, anywhere a maintainer can reasonably link the behavior to OCP).
- Direct, technical voice (see `AGENTS.md`) is the project's house style. Direct is not a license to be hostile.

## Reporting

Report violations to me@chrisellis.dev. Reports are confidential. The maintainer adjudicates and acts proportionately: private warning, comment edit or removal, issue or PR ban, repository ban.

If a violation involves the maintainer, escalate to a third party of your choosing and notify the maintainer in writing.

## Scope

This code applies to anyone participating in the project: maintainers, contributors, issue commenters, and anyone interacting with OCP-authored content. It applies whether you are paid, volunteering, or running OCP against your own repo.
